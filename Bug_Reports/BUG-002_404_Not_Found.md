# BUG-002 — Link opens 404 Not Found

**Severity:** Medium  
**Priority:** P2  
**Environment:** Chrome (Desktop)

## Description
Clicking certain category links results in a 404 Not Found.

## Steps to Reproduce
1. Go to https://arngren.net/
2. Click a category link (example: “Elektriske Biler”)

## Expected Result
Category page loads successfully.

## Actual Result
Page returns 404 Not Found.

## Notes
Broken link or incorrect routing.
